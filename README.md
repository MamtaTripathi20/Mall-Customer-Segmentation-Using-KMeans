# Mall Customer Segmentation Using K-Means Clustering

# Project Overview

This project demonstrates customer segmentation using the K-Means Clustering algorithm. The objective is to group customers based on their annual income and spending behavior to help businesses understand customer purchasing patterns.

# Dataset

- Dataset: Mall Customers Dataset
- Total Records: 200
- Features:
  - CustomerID
  - Gender
  - Age
  - Annual Income (k$)
  - Spending Score (1-100)

# Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

# Project Workflow

- Import Libraries
- Load Dataset
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Customer Segmentation using K-Means
- Cluster Visualization

# Visualizations

- Distribution Plots
- Count Plot
- Violin Plot
- Age Distribution
- Spending Score Analysis
- Customer Clusters

# Results

The K-Means algorithm successfully grouped customers into different clusters based on Annual Income and Spending Score, helping identify distinct customer segments for business decision-making.
